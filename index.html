Exercice d'algorithme.

Exercice de l'échiquier.

Pour cet algorithme, vous devez placer les différentes pièces noires ( p (pion), f (fou), d (dame), r (roi), t (tour), c (cavalier)) sur les cases présentes de l'échiquier. Ensuite vous placerez le/les cavalier(s) blanc(s) (C (cavalier)). Vous verrez alors, si celui-ci est capable de capturer une pièce noire.
